演算法列表

🫧 氣泡排序

Bubble Sort
最佳時間
O(n)
平均時間
O(n²)
最差時間
O(n²)
空間複雜度
O(1)

最簡單的排序演算法。通過重複比較相鄰元素並交換，將最大值逐步「浮」到陣列末尾。適合初學者理解排序的基本概念。

👆 選擇排序

Selection Sort
最佳時間
O(n²)
平均時間
O(n²)
最差時間
O(n²)
空間複雜度
O(1)

每次從未排序部分選擇最小元素，放在已排序部分的末尾。性能比氣泡排序更穩定，交換次數較少。

📌 插入排序

Insertion Sort
最佳時間
O(n)
平均時間
O(n²)
最差時間
O(n²)
空間複雜度
O(1)

類似整理撲克牌的過程。將元素一個一個插入已排序的序列中。對小規模陣列或接近排序的陣列效率高。
