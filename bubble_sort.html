📖 什麼是氣泡排序法？

  • 從陣列的第一個元素開始，兩兩比較相鄰元素，若左邊 > 右邊則交換。
  • 每一輪結束後，當前未排序部分的最大值會「浮」到最右邊，如同氣泡上升。
  • 重複 n-1 輪後，整個陣列完成排序。
  • 優化技巧：若某一輪沒有任何交換，代表已排序完成，可提早結束。

📊 時間與空間複雜度

情況時間複雜度空間複雜度說明

💻 C 語言實作（優化版）

#include <stdio.h>
#include <stdbool.h>

void bubbleSort(int arr[], int n) {
    int i, j, temp;
    bool swapped;

    for (i = 0; i < n - 1; i++) {
        swapped = false;
        for (j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                temp       = arr[j];
                arr[j]     = arr[j + 1];
                arr[j + 1] = temp;
                swapped    = true;
            }
        }
        // 若本輪沒有交換，陣列已排序完成
        if (!swapped) break;
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    bubbleSort(arr, n);

    for (int i = 0; i < n; i++)
        printf("%d ", arr[i]);
    return 0;
}
